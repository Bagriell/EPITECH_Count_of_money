# Count Of Money

## Api: Nodejs + Express + SequelizeORM
## Front: Vuejs 2

### Api
`create postgres database`

`cd api && npm i`

`create .env with 
POSTGRES_USER= ???
POSTGRES_PASSWORD= ???
POSTGRES_DB= ???`

`change config/db.config.js HOST with your own ip address`

### Front

`cd front && npm i`

### Docker

`sudo docker-compose up --build`
